M5 · Skill 评测

系统化评估 Skill 质量，驱动自进化迭代

模块概述

Skill 评测模块提供标准化的 AI 能力评估流程。通过 5 步向导式操作，用户可以配置评测智能体、 设置工具环境、准备评测数据集、定义评测指标，最终启动自动化评测任务。 支持跨平台 Skill（Friday / AIbuddy / ZeroAgent）的统一评测。

用户故事

作为Skill 开发者，我希望能对 Skill 进行系统化评测，以便验证其质量并持续优化。
作为业务运营，我希望在上线前评测 Skill 的效果，以便确保数字员工的服务质量。
作为技术负责人，我希望基于评测结果对比不同版本的 Skill，以便做出数据驱动的迭代决策。

5步评测流程

① 智能体配置
→
② 工具配置
→
③ 评测集
→
④ 评测指标
→
⑤ 启动评测

各步骤详情

① 智能体配置

  • 智能体类型选择：Skill / Agent（Radio）
  • Skill 选择：搜索式下拉框，输入名称搜索
  • 版本指定：下拉选择目标版本
  • 支持来源：本平台 / Friday / AIbuddy / ZeroAgent
  • 快捷操作：新建 Skill / 前往 Friday 创建

② 工具配置

  • 配置评测过程中 Skill 可调用的工具
  • 工具权限管理（读/写/执行）
  • 环境变量设置
  • API 端点配置

③ 评测集

  • 上传或选择评测数据集
  • 测试用例管理（输入-期望输出对）
  • 支持批量导入/导出
  • 数据集版本管理

④ 评测指标

  • 预设指标：准确率、召回率、F1
  • 自定义指标定义
  • 评分维度：完整性、相关性、安全性
  • 基准线设置

⑤ 启动评测

  • 配置确认与预览
  • 一键启动评测任务
  • 实时进度展示
  • 评测结果报告生成

验收标准

  • ✅ 5 步向导完整流转，每步数据正确传递到下一步
  • ✅ 支持选择跨平台 Skill（Friday/AIbuddy/ZeroAgent）
  • ✅ 评测任务成功执行并生成结果报告
  • ✅ 「下一步」和「取消」按钮交互正确
  • ✅ Skill 搜索支持模糊匹配

模块概述

费用查询模块提供租户级别的 AI 服务成本分析能力，包含用量汇总和下钻分析两大视图。 支持按时间范围查看总费用、Token 消耗（输入/输出/缓存命中）、模型级别的费用构成， 以及按数字员工、FridayKey、用户 MIS 等多维度的下钻分析。

用户故事

作为租户管理员，我希望能查看整体 AI 费用和 Token 用量趋势，以便掌握成本大盘。
作为业务负责人，我希望能按数字员工维度下钻分析费用，以便识别高消耗场景并优化。
作为财务人员，我希望能导出费用明细报表，以便进行成本核算和预算管理。

Tab 1：📊 用量汇总

📅 时间筛选

  • 快捷选项：近3天 / 近7天
  • 自定义日期范围选择器
  • 最后更新时间戳显示
  • 手动刷新按钮

📊 4 大 KPI 卡片

  • 💰 总费用（CNY）— 蓝色边框
  • 📥 输入 Token — 橙色边框
  • 📤 输出 Token — 绿色边框
  • ⚡ 缓存命中 Token — 紫色边框

📈 每日费用趋势图

  • 折线/柱状图展示每日费用变化
  • 支持悬浮查看具体数值
  • 空状态提示：「暂无趋势数据」

🥧 费用构成表

  • 表格列：模型、金额(¥)、占比

  • 空状态提示：「暂无数据」

Tab 2：🔍 下钻分析
